# `CRITICAL`

## ShellShock - Remote Code Execution
- **End-Point**: http://10.64.162.78/cgi-bin/test.cgi/debug.cgi
- **영향**: 원격 코드 실행을 통해 무단 접근, 데이터 탈취 및 시스템 침해가 발생할 수 있음.
- **설명**: GNU Bash 4.3 이하 버전에서 환경 변수 값 내 함수 정의 뒤에 오는 문자열을 처리할 때 발생하는 취약점임(CVE-2014-6271).
- **근거**: HTTP 헤더(Cookie, Referer 등)에 함수 정의 구문과 시스템 명령어(/bin/cat /etc/passwd)를 포함하여 전송함.
- **대응**: -
- **조치**: -

## Visual Tools DVR VX16 4.2.28.0 - Unauthenticated OS Command Injection
- **End-Point**: http://10.64.162.78/cgi-bin/test.cgi/cgi-bin/slogin/login.py
- **영향**: 무단 원격 코드 실행으로 이어져 시스템의 기밀성, 무결성, 가용성을 침해할 수 있음
- **설명**: Visual Tools DVR VX16 4.2.28.0 장비에서 인증되지 않은 원격 공격자가 명령 주입 공격을 수행할 수 있음
- **근거**: User-Agent 헤더에 쉘 명령을 주입하여 /etc/passwd 파일을 조회하는 cURL 요청으로 확인됨
- **대응**: -
- **조치**: -

## Spring Security BCrypt Truncation & ACL Bypass Detection
- **End-Point**: http://10.64.162.78/cgi-bin/test.cgi/admin/search
- **영향**: N/A
- **설명**:

# `MEDIUM`

## OpenSSH Terrapin Attack - Detection
- **End-Point**: 10.64.162.78:22
- **영향**: 클라이언트와 서버 연결 시 일부 보안 기능이 다운그레이드되거나 비활성화될 수 있음.
- **설명**: OpenSSH 9.6 미만 등에서 핸드셰이크 및 시퀀스 번호 처리 미흡으로 무결성 검사를 우회하는 Terrapin 공격 취약점임.
- **근거**: 제공된 증거 및 재현 정보 없음.
- **대응**: OpenSSH 9.6 미만, PuTTY 0.80 미만 등 영향을 받는 소프트웨어 버전 사용 여부 점검.
- **조치**: OpenSSH 9.6 이상 또는 각 벤더별(Dropbear, PuTTY 등) 취약점이 해결된 최신 버전으로 업데이트.
