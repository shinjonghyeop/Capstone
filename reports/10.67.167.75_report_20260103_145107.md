# `CRITICAL`

## Apache ActiveMQ Fileserver - Arbitrary File Write  
- **엔드포인트**: http://10.67.167.75:8161/fileserver/37VcjAdfl4YDajAzoofyBM9nIcW.txt  
- **영향**:  
  - 원격 코드 실행 가능성 있음  
- **설명**:  
  Apache ActiveMQ 5.x(버전 5.14.0 이전)에서 HTTP PUT 후 HTTP MOVE 요청을 통해 임의 파일 업로드 및 이동이 가능하며, 이로 인해 서버에 악성 코드 실행이 발생할 수 있음  
- **근거**:  
  재현 예시 제공됨 (curl 명령어 사용)  
- **대응**:  
  - ActiveMQ 버전을 5.14.0 이상으로 업데이트  
- **조치**:  
  - 파일 업로드 요청 차단 정책 적용 (예: allowlist 기반 IP 화이트리스트)
