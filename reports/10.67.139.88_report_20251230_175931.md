# `CRITICAL`
## ShellShock - Remote Code Execution
- **End-Point**: http://10.67.139.88/cgi-bin/test.cgi
- **영향**: 원격 코드 실행을 통해 시스템 무단 접근 및 데이터 탈취 가능.
- **설명**: Bash 4.3 이하 버전에서 환경 변수 처리가 미흡하여 임의 명령 실행이 가능함.
- **근거**: `Cookie: () { ignored; }; echo Content-Type: text/html; ... /bin/cat /etc/passwd`
- **대응**: 운영체제 패키지 관리자를 통해 Bash를 최신 버전으로 업데이트.
- **조치**: 불필요한 CGI 스크립트 비활성화 및 웹 방화벽(WAF) 탐지 규칙 적용.

## Visual Tools DVR VX16 - Unauthenticated OS Command Injection
- **End-Point**: http://10.67.139.88/cgi-bin/test.cgi/cgi-bin/slogin/login.py
- **영향**: 인증되지 않은 원격 공격자가 시스템 제어권을 탈취하여 코드 실행 가능.
- **설명**: Visual Tools DVR VX16의 로그인 스크립트에서 입력값 검증 부재로 발생.
- **근거**: `User-Agent: () { :; }; echo ; echo ; /bin/cat /etc/passwd`
- **대응**: 해당 DVR 펌웨어를 보안 패치가 적용된 최신 버전으로 업데이트.
- **조치**: 외부에서의 관리 페이지 접근을 차단하고 VPN 등을 통해서만 접근 허용.
