Prompt Evaluation Metrics (generate_reports.py)

Goal
Measure how well the report output follows the prompt rules and input data.

Metrics (binary or percent)
1) format_heading_only_1_2
   - 1 if only # and ## headings are used (no ### or deeper)
2) format_no_html
   - 1 if no "<tag>" appears
3) format_no_emoji
   - 1 if no emoji or non-ASCII symbols appear
4) format_bullets_short
   - percent of bullet lines <= 90 chars
5) format_bullets_count
   - 1 if each section has <= 6 bullets
6) format_tables_present
   - 1 if Summary and Stats tables exist
7) format_summary_table_match
   - 1 if Summary table rows match provided summary_table (same order, same rows)
8) format_stats_table_match
   - 1 if Severity + Category + Endpoint tables match provided tables
9) content_no_hallucination
   - 1 if all findings titles exist in input
10) content_evidence_only
   - 1 if evidence text is only from provided evidence/curl/request/response
11) content_no_low_in_findings
   - 1 if Low severity is excluded from the findings section
12) content_severity_groups_only_when_present
   - 1 if severity headings appear only when items exist
13) content_endpoints_aggregated
   - percent of grouped findings with merged endpoints list
14) coverage_findings
   - percent of unique Critical/High/Medium items present vs input

Scoring (example)
- Weighted score = sum(weight_i * metric_i)
  format_heading_only_1_2: 0.10
  format_no_html:          0.05
  format_no_emoji:         0.05
  format_bullets_short:    0.05
  format_bullets_count:    0.05
  format_tables_present:   0.10
  format_summary_table_match: 0.10
  format_stats_table_match:   0.10
  content_no_hallucination:   0.15
  content_evidence_only:      0.10
  content_no_low_in_findings: 0.05
  content_severity_groups_only_when_present: 0.05
  content_endpoints_aggregated: 0.03
  coverage_findings: 0.02

Pass/Fail Gates (example)
- Gate A (format): metrics 1-8 must all be 1
- Gate B (truthfulness): metrics 9-11 must all be 1
- Gate C (coverage): coverage_findings >= 0.85
