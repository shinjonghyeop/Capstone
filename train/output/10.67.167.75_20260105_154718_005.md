## Unencrypted Channels
- **End-Point**: `/admin/`, `/` 외 1개
- **영향**: 암호화되지 않은 HTTP 통신을 통해 데이터가 평문으로 전송되므로, 네트워크 스니핑이나 중간자 공격(MitM)에 의해 사용자 자격 증명 및 민감 정보가 탈취될 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션이 구성되어 있지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 처리됨.
- **근거**: `curl "http://10.67.167.75:8161/admin/"` 실행 시 HTTP 프로토콜을 통한 접근이 허용됨을 확인.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉트하도록 설정하고, HSTS(HTTP Strict Transport Security) 헤더를 적용하여 브라우저가 항상 HTTPS를 사용하도록 강제해야 함.
- **조치**: 웹 서버(Nginx, Apache 등) 또는 로드 밸런서의 설정 파일에서 80번 포트의 요청을 443번 포트로 리다이렉트하는 규칙을 추가하고 유효한 SSL/TLS 인증서를 적용함.
