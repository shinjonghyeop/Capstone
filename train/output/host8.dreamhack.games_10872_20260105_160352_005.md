## Inconsistent Redirection
- **End-Point**: `/`
- **영향**: 리다이렉션 응답 본문에 포함된 예기치 않은 HTML 요소(링크, 폼 등)를 통해 사용자가 의도하지 않은 정보에 노출되거나 피싱 공격 등에 악용될 수 있음.
- **설명**: 서버가 3xx 리다이렉션 응답을 보낼 때, 표준적인 리다이렉션 메시지 범위를 벗어나 링크나 폼과 같은 예기치 않은 HTML 본문이 포함되어 발생하는 취약점임.
- **근거**: `curl "http://host8.dreamhack.games:10872/"` 명령을 실행하여 3xx 응답 본문에 비정상적인 HTML 코드가 포함되어 있는지 확인함.
- **대응**: 리다이렉션 응답 시 본문 내용을 비우거나, 표준에 정의된 최소한의 안내 문구만을 포함하도록 서버 설정을 변경해야 함.
- **조치**: HTTP 응답 헤더의 `Location`을 통한 리다이렉션 처리 시 본문에 민감한 데이터나 실행 가능한 HTML 태그가 포함되지 않도록 소스 코드를 수정함.
