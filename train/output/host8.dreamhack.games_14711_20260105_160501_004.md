## Unencrypted Channels
- **End-Point**: /, /flag 외 1개
- **영향**: 암호화되지 않은 평문 통신을 통해 데이터가 노출되거나 중간자 공격(MITM)에 의한 세션 탈취 및 정보 변조가 발생할 수 있습니다.
- **설명**: 해당 호스트에 HTTPS 리다이렉션 설정이 적용되어 있지 않아 모든 HTTP 요청과 응답이 암호화되지 않은 상태로 전송됩니다.
- **근거**: `curl "http://host8.dreamhack.games:14711/"` 명령을 실행하여 HTTP 프로토콜로 서비스에 직접 접근이 가능함을 확인하였습니다.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉트하고 HSTS(HTTP Strict Transport Security) 헤더를 적용하여 보안 채널을 강제해야 합니다.
- **조치**: 웹 서버 설정에서 80번 포트 접근 시 443번 포트로 리다이렉트하도록 구성하고 유효한 SSL/TLS 인증서를 설치하여 통신을 암호화하십시오.
