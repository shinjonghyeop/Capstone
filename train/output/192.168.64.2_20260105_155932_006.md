## Unencrypted Channels
- **End-Point**: `/`, `/login` 외 1개
- **영향**: 공격자가 네트워크 트래픽을 가로채는 중간자 공격(MitM)을 통해 로그인 자격 증명, 세션 쿠키 등 민감한 정보를 평문으로 탈취할 수 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션이 설정되어 있지 않아 모든 HTTP 요청이 암호화되지 않은 평문 상태로 전송됨.
- **근거**: `curl "http://192.168.64.2:5000/"` 실행 시 HTTP 프로토콜을 통한 평문 접근이 허용됨을 확인.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉션하도록 서버 설정을 변경하고 HSTS(HTTP Strict Transport Security) 보안 헤더를 적용해야 함.
- **조치**: 웹 서버(Nginx, Apache 등) 설정에서 80번 포트로 들어오는 요청을 443번 포트(HTTPS)로 리다이렉트하고 신뢰할 수 있는 SSL/TLS 인증서를 적용함.
