## Unencrypted Channels
- **End-Point**: `/cgi-bin/test.cgi`, `/text/css` 외 3개
- **영향**: 암호화되지 않은 통신 채널을 통해 데이터가 전송되어 중간자 공격(MITM) 시 사용자 정보 및 민감한 데이터가 평문으로 노출되거나 변조될 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션 설정이 존재하지 않아 모든 HTTP 요청이 암호화되지 않은 평문 상태로 처리되고 있음.
- **근거**: `curl "http://10.67.139.88/cgi-bin/test.cgi"` 명령을 실행하여 HTTP 평문 통신이 보안 경고 없이 성공적으로 수행됨을 확인함.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉션하도록 서버 설정을 변경하고, HSTS(HTTP Strict Transport Security)를 적용하여 보안 수준을 강화해야 함.
- **조치**: 웹 서버(Nginx, Apache 등) 설정 파일에서 80 포트로 들어오는 요청에 대해 443 포트로의 리다이렉션(301 Moved Permanently) 규칙을 추가함.
