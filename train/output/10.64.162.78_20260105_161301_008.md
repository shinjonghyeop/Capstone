## Unencrypted Channels
- **End-Point**: `/cgi-bin/test.cgi`
- **영향**: 데이터가 암호화되지 않은 평문으로 전송되어 중간자 공격(MITM)을 통한 정보 유출 및 데이터 변조의 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션 설정이 되어 있지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 처리됨.
- **근거**: `curl "http://10.64.162.78/cgi-bin/test.cgi"` 명령을 통해 HTTP 프로토콜로의 직접적인 접근 및 통신이 가능함을 확인.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉션하도록 서버 설정을 구성하고, HSTS(HTTP Strict Transport Security)를 적용함.
- **조치**: 웹 서버 설정에서 80번 포트(HTTP)로 인입되는 모든 트래픽을 443번 포트(HTTPS)로 전환하고 SSL/TLS 인증서를 적용하여 통신 구간을 암호화함.
