## Inconsistent Redirection
- **End-Point**: `/`
- **영향**: 리다이렉션 응답 바디에 포함된 HTML 콘텐츠가 사용자에게 노출되어 피싱 공격에 이용되거나 의도치 않은 정보가 유출될 수 있음.
- **설명**: HTTP 3xx 리다이렉션 응답 시 본문에 예상치 못한 HTML 코드(링크, 폼 등)가 포함되어 있는 상태임.
- **근거**: `curl "http://host8.dreamhack.games:19373/"` 명령 실행 시 3xx 응답과 함께 HTML 본문이 반환됨을 확인.
- **대응**: 리다이렉션 응답 발생 시 HTML 바디를 제거하거나 표준에 맞는 최소한의 정보만 포함하도록 설정함.
- **조치**: 서버 환경 설정 또는 애플리케이션 코드를 수정하여 3xx 상태 코드 반환 시 불필요한 HTML 본문이 생성되지 않도록 조치함.
