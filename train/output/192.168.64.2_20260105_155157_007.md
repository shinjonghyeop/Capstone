## Unencrypted Channels
- **End-Point**: `/www/SQL/sql1.php`, `/www/FileUpload/fileupload1.php` 외 8개
- **영향**: 네트워크 통신 시 데이터가 암호화되지 않은 평문으로 전송되어 중간자 공격(MITM)을 통한 민감 정보 노출 및 데이터 변조 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션 설정이 존재하지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 처리됨.
- **근거**: `curl "http://192.168.64.2:9991/www/SQL/sql1.php"` 실행 시 HTTP 프로토콜을 통해 평문 데이터가 송수신됨을 확인.
- **대응**: 전송 계층 보안(TLS/SSL)을 적용하여 모든 통신 과정을 암호화하고 HTTP 요청을 HTTPS로 강제 전환해야 함.
- **조치**: 웹 서버 설정에서 80번 포트(HTTP) 접속 시 443번 포트(HTTPS)로 리다이렉트하도록 설정하고 HSTS(HTTP Strict Transport Security) 보안 헤더를 적용함.
