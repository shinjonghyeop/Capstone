## Comprehensive XSS Detection (All Vectors)
- **End-Point**: `http://localhost:9991/www/XSS/XSS_level2.php`, `http://localhost:9991/www/XSS/XSS_level1.php` 외 3개
- **영향**: 공격자가 사용자의 브라우저에서 임의의 스크립트를 실행하여 세션 토큰 탈취, 웹사이트 변조, 피싱 공격 등을 수행할 수 있음
- **설명**: 다양한 우회 기법, 폴리글랏, WAF 우회 및 고급 페이로드를 결합하여 애플리케이션의 XSS 방어 메커니즘을 점검하는 포괄적인 탐지 템플릿임
- **근거**: `q` 파라미터에 스크립트 페이로드를 주입하여 취약점을 재현함: `curl -X 'GET' -d '' -H 'Accept: */*' -H 'Accept-Language: en' -H 'User-Agent: Mozilla/5.0 (ZZ; Linux i686; rv:125.0) Gecko/20100101 Firefox/125.0' 'http://localhost:9991/www/XSS/XSS_level2.php?q=<script>alert({{1}})</script>'`
- **대응**: 모든 사용자 입력값에 대해 화이트리스트 기반의 검증을 수행하고 출력 시 HTML 엔티티 인코딩을 적용해야 함
- **조치**: 서버 측에서 데이터를 렌더링할 때 `htmlspecialchars` 등의 함수를 사용하여 특수 문자를 변환하고 `Content-Security-Policy (CSP)`를 설정하여 인라인 스크립트 실행을 차단함
