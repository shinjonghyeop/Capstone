## Django Debug Page - Cross-Site Scripting  
- **End-Point**: http://localhost:9991/www/XSS/XSS_level1.php/create_user/  
- **영향**:  
  - Successful exploitation 가능성 있음  
  - 피해자 브라우저 내에서 스크립트 실행 → 세션 하이재킹, 페이지 변조, 정보 유출 등  
- **설명**:  
  Django 1.10.x~1.11.x에서 DEBUG 플래그가 활성화될 때 HTML autoescaping이 비활성화된 템플릿을 통해 발생하는 취약점.  
- **근거**:  
  - 재현 가능한 테스트 사례 제공 (curl command 포함)  
  - CVE CVE-2017-12794에서 명시된 기술적 결함 확인  
- **대응**:  
  - Django 버전을 CVE 영향 범위 미만으로 업데이트 (예: 1.11.6 이상)  
  - DEBUG 플래그 비활성화 (production 환경에서 필수)  
- **조치**:  
  - 즉시 패치 적용 권장
