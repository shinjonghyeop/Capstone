## Inconsistent Redirection
- **End-Point**: `/login`
- **영향**: 3xx 리다이렉션 응답 본문에 예기치 않은 HTML 코드가 포함되어 정보 노출이나 피싱 등 2차 공격에 활용될 수 있음.
- **설명**: 3xx 리다이렉션 상태 코드 반환 시 HTTP 응답 본문(Body) 영역에 링크나 폼과 같은 예상치 못한 HTML 요소가 포함되어 있는 취약점임.
- **근거**: `curl "http://192.168.64.2:5000/login" -e "http://192.168.64.2:5000/" -d "username=alice&password=Letm3in_"` 명령 실행을 통해 리다이렉션 응답 시 HTML 본문이 반환되는 것을 확인 가능.
- **대응**: 리다이렉션 응답 시 본문 내용을 제거하거나 표준에 정의된 최소한의 정보만 포함하도록 설정하고, 리다이렉션 대상 URL에 대한 검증을 수행함.
- **조치**: 서버 환경 설정 또는 애플리케이션 소스 코드를 수정하여 3xx 응답 시 불필요한 HTML 바디 데이터가 생성되지 않도록 조치함.
