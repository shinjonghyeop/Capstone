## Unencrypted Channels
- **End-Point**: `/`, `/memo` 외 3개
- **영향**: 암호화되지 않은 평문 통신을 통해 데이터가 전송되므로 중간자 공격(MITM)에 의한 민감 정보 도청 및 세션 탈취가 가능함.
- **설명**: 해당 호스트에 HTTPS 리다이렉션이 구성되어 있지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 처리됨.
- **근거**: `curl "http://127.0.0.1:8000/"` 실행 시 HTTP 프로토콜을 통해 서비스에 접근 가능한 것을 확인함.
- **대응**: 모든 HTTP 연결을 HTTPS로 강제 전환하고, HSTS(HTTP Strict Transport Security) 헤더를 적용하여 보안 수준을 강화해야 함.
- **조치**: 웹 서버 설정에서 80번 포트로 들어오는 모든 요청을 443번 포트(HTTPS)로 301 리다이렉트하도록 설정하고 신뢰할 수 있는 SSL/TLS 인증서를 설치함.
