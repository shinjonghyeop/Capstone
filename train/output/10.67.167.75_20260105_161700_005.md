## Unencrypted Channels
- **End-Point**: `/admin/`, `/` 외 1개
- **영향**: 통신 내용이 평문으로 전송되어 네트워크 상의 공격자가 데이터를 가로채거나 세션 토큰 및 민감한 정보를 탈취할 수 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션이 설정되어 있지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 처리됨.
- **근거**: `curl "http://10.67.167.75:8161/admin/"` 명령을 통해 암호화되지 않은 HTTP 프로토콜로 서비스 접근이 가능함을 확인.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉션하도록 서버 설정을 변경하고 SSL/TLS 인증서를 적용해야 함.
- **조치**: 웹 서버 설정에서 80 포트(HTTP) 접속 시 443 포트(HTTPS)로 301 Redirect를 수행하도록 구성하고 HSTS 헤더를 활성화함.
