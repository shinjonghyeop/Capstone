## Inconsistent Redirection
- **End-Point**: `/news.php`, `/dashboard.php`
- **영향**: 리다이렉션 응답 본문에 포함된 HTML 요소(링크, 폼 등)를 통해 의도하지 않은 정보가 노출되거나, 공격자가 애플리케이션의 구조를 파악하는 데 악용될 수 있음.
- **설명**: 서버가 3xx 상태 코드를 반환하여 페이지를 이동시킬 때, 응답 본문에 예상치 못한 HTML 코드(링크 또는 폼 태그 등)가 포함되어 발생하는 현상임.
- **근거**: `curl "http://10.66.164.167/news.php"` 명령을 통한 재현 결과 확인.
- **대응**: 리다이렉션 응답 시 HTTP 본문 내용을 제거하고 `Location` 헤더만을 사용하여 안전하게 페이지를 이동시켜야 함.
- **조치**: 서버 측 코드에서 3xx 응답을 생성할 때 본문(Body)에 HTML 콘텐츠가 포함되지 않도록 설정을 변경하고, 필요 시 본문을 빈 상태로 전송하도록 수정함.
