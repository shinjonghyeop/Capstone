## OpenSSH Terrapin Attack - Detection
- **End-Point**: `10.66.164.167:22`
- **영향**: SSH 전송 프로토콜의 무결성 검사를 우회하여 특정 보안 기능을 저하시키거나 비활성화할 수 있음.
- **설명**: OpenSSH 9.6 미만 버전 등에서 발생하는 Terrapin 공격(`CVE-2023-48795`)으로, SSH 핸드셰이크 과정에서 시퀀스 번호를 조작하여 무결성 검사를 우회함. `chacha20-poly1305@openssh.com` 및 `*-etm@openssh.com` 알고리즘을 사용할 때 특히 취약하며, 공격자는 확장 협상 메시지 등에서 일부 패킷을 누락시켜 연결 보안 수준을 낮출 수 있음.
- **근거**: -
- **대응**: 취약한 암호화 알고리즘(`chacha20-poly1305`, `EtM` 방식의 MAC) 사용 여부를 확인하고 보안 패치 적용을 준비함.
- **조치**: OpenSSH를 9.6 이상의 최신 버전으로 업데이트하거나, 서버 설정에서 취약한 알고리즘을 제거하고 `strict key exchange` 기능을 지원하도록 구성함.
