# `CRITICAL`
## [ShellShock - Remote Code Execution]
- **End-Point**: http://10.64.162.78/cgi-bin/test.cgi/debug.cgi
- **영향**: Remote code execution can lead to unauthorized access and system compromise.
- **설명**: GNU Bash through 4.3 processes trailing strings after function definitions.
- **근거**: `curl -X 'GET' -H 'Cookie: () { ignored; }; ... /bin/cat /etc/passwd'`
- **대응**: CVE-2014-6271 취약점이 해결된 Bash 버전으로 업데이트.
- **조치**: 운영체제 패치 적용 및 비정상적인 환경변수 요청 차단 설정.

## [Visual Tools DVR VX16 4.2.28.0 - Unauthenticated OS Command Injection]
- **End-Point**: http://10.64.162.78/cgi-bin/test.cgi/cgi-bin/slogin/login.py
- **영향**: Unauthorized remote code execution compromising system confidentiality.
- **설명**: Visual Tools DVR VX16 4.2.28.0 allows unauthenticated command injection.
- **근거**: `curl -H 'User-Agent: () { :; }; ... /bin/cat /etc/passwd'`
- **대응**: Visual Tools DVR 펌웨어를 보안 패치가 적용된 최신 버전으로 업데이트.
- **조치**: 장비에 대한 외부 접근을 제한하고 벤더 보안 권고안에 따라 패치 수행.

## [Spring Security BCrypt Truncation & ACL Bypass Detection]
- **End-Point**: http://10.64.162.78/cgi-bin/test.cgi/admin/search
- **영향**: Potential ACL bypass leading to unauthorized access.
- **설명**: Access /admin/search using a truncated password (CVE-2025-22228).
- **근거**: `curl -X 'POST' -d '{"apiKey": "...", "password": "..."}'`
- **대응**: BCrypt Truncation 이슈가 해결된 Spring Security 버전으로 업그레이드.
- **조치**: 패스워드 길이 검증 로직을 강화하고 프레임워크 보안 설정 검토.

# `MEDIUM`
## [OpenSSH Terrapin Attack - Detection]
- **End-Point**: 10.64.162.78:22
- **영향**: Bypass of integrity checks during SSH extension negotiation.
- **설명**: SSH transport protocol allows omitting packets from extension negotiation.
- **근거**: N/A
- **대응**: OpenSSH 9.6 이상으로 업데이트하거나 영향을 받는 암호화 알고리즘 비활성화.
- **조치**: SSH 서버 구성을 점검하고 Strict KEX 모드 활성화 검토.
