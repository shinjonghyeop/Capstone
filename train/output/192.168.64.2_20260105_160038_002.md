## Comprehensive XSS Detection (All Vectors)
- **End-Point**: `http://192.168.64.2:3000/`
- **영향**: 사용자의 브라우저에서 악의적인 스크립트가 실행되어 세션 쿠키 탈취, 페이지 변조, 피싱 공격 및 사용자 권한 오용이 발생할 수 있음.
- **설명**: 다양한 우회 기법, 폴리글롯(Polyglot), WAF 우회 패턴 및 고급 페이로드를 결합하여 애플리케이션 내의 크로스 사이트 스크립팅(XSS) 취약점을 종합적으로 탐지함.
- **근거**: 다음 재현 명령을 통해 `<script>alert({{1}})</script>` 페이로드가 파라미터를 통해 전달됨을 확인: `curl -X 'GET' -d '' -H 'Accept: */*' -H 'Accept-Language: en' -H 'User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/90.0.4430.93 Safari/537.36' 'http://192.168.64.2:3000/?param=<script>alert({{1}})</script>'`
- **대응**: 모든 사용자 입력값에 대해 화이트리스트 기반의 검증을 수행하고, 웹 페이지 출력 시 HTML 엔티티 인코딩을 적용하여 스크립트가 실행되지 않도록 처리함.
- **조치**: `Content-Security-Policy (CSP)` 헤더를 적용하여 신뢰할 수 없는 스크립트 실행을 제한하고, 민감한 쿠키에 `HttpOnly` 속성을 부여하여 스크립트를 통한 접근을 차단함.
