## MIME Type Confusion
- **End-Point**: `/admin/`, `/` 외 1개
- **영향**: 브라우저가 리소스의 MIME 타입을 추측하여 실행하는 MIME 스니핑 공격에 노출될 수 있으며, 이는 교차 사이트 스크립팅(XSS) 등의 보안 위협으로 이어질 수 있음.
- **설명**: HTTP 응답 헤더에 `X-Content-Type-Options: nosniff` 설정이 누락되어 있어, 브라우저가 Content-Type 헤더에 명시된 타입과 다른 방식으로 응답 데이터를 해석할 위험이 있음.
- **근거**: ```curl "http://10.67.167.75:8161/admin/"``` 명령 실행을 통한 취약점 재현 확인.
- **대응**: 서버의 모든 HTTP 응답 헤더에 `X-Content-Type-Options: nosniff` 구문을 추가하여 브라우저가 응답 본문의 내용을 임의로 추측하지 않도록 설정함.
- **조치**: Nginx, Apache 등 웹 서버 설정 파일이나 애플리케이션의 보안 미들웨어를 수정하여 해당 보안 헤더가 일괄 적용되도록 조치함.
