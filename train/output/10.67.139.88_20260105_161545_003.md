## OpenSSH Terrapin Attack - Detection
- **End-Point**: `10.67.139.88:22`
- **영향**: SSH 연결의 무결성 검사 우회 및 보안 기능 강등(Downgrade)을 통한 데이터 조작 가능성
- **설명**: 특정 OpenSSH 확장 기능이 포함된 SSH 전송 프로토콜에서 발생하는 Terrapin 공격(`CVE-2023-48795`)으로, 핸드셰이크 과정에서 시퀀스 번호를 잘못 처리하여 공격자가 중간에서 패킷을 누락시키거나 보안 협상 메시지를 조작할 수 있음. 특히 `chacha20-poly1305@openssh.com` 및 `Encrypt-then-MAC(EtM)` 알고리즘을 사용하는 환경에서 취약하며, 이를 통해 클라이언트와 서버 간의 보안 설정이 약화될 수 있음.
- **근거**: -
- **대응**: OpenSSH 버전을 `9.6` 이상으로 업데이트하여 "strict key exchange" 기능을 적용하고 취약한 암호화 알고리즘 사용 여부를 점검함.
- **조치**: `sshd_config` 설정에서 취약한 알고리즘인 `chacha20-poly1305@openssh.com` 및 `*-etm@openssh.com` MAC 알고리즘을 제외하거나, 최신 보안 패치가 적용된 클라이언트 및 서버 소프트웨어를 사용함.
