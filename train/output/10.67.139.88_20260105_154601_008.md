## Inconsistent Redirection
- **End-Point**: `/cgi-bin`
- **영향**: 리다이렉션 응답에 포함된 HTML 본문이 사용자에게 노출되어 피싱 공격에 악용되거나 의도치 않은 정보가 유출될 위험이 있습니다.
- **설명**: HTTP 3xx 리다이렉션 응답 시 `Location` 헤더 외에 링크나 폼과 같은 예상치 못한 HTML 본문이 포함되어 있는 상태입니다.
- **근거**: `curl "http://10.67.139.88/cgi-bin"` 명령을 통한 재현 결과 확인.
- **대응**: 리다이렉션 응답 시 HTML 본문 내용을 제거하거나 최소한의 표준 메시지만 포함하도록 서버 설정을 변경해야 합니다.
- **조치**: 웹 서버 또는 애플리케이션 설정에서 3xx 상태 코드 반환 시 자동으로 생성되는 HTML 템플릿을 수정하거나 비활성화합니다.
