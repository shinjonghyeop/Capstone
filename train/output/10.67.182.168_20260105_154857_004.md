## MIME Type Confusion
- **End-Point**: `/admin/`, `/text/css` 외 1개
- **영향**: 브라우저가 콘텐츠의 MIME 타입을 임의로 추측(Sniffing)하여 실행함으로써 크로스 사이트 스크립팅(`XSS`) 등 악성 공격에 노출될 수 있음.
- **설명**: 서버 응답 헤더에 `X-Content-Type-Options`가 설정되지 않아 브라우저가 `Content-Type` 헤더를 무시하고 데이터를 직접 해석할 수 있는 상태임.
- **근거**: `curl "http://10.67.182.168:8161/admin/"` 명령을 통한 응답 결과에서 `X-Content-Type-Options` 헤더가 확인되지 않음.
- **대응**: 모든 HTTP 응답 헤더에 `X-Content-Type-Options: nosniff`를 추가하여 브라우저의 MIME 스니핑 기능을 비활성화해야 함.
- **조치**: 웹 서버(Nginx, Apache 등) 또는 애플리케이션의 설정에서 `X-Content-Type-Options` 보안 헤더를 전역적으로 적용함.
