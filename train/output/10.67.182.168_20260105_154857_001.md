## Apache ActiveMQ Fileserver - Arbitrary File Write
- **End-Point**: `http://10.67.182.168:8161/fileserver/37bqUJvN7ZHqMwVRB5KXFZipYuB.txt`
- **영향**: 공격자가 서버에 임의의 파일을 작성할 수 있으며, 이는 최종적으로 원격 코드 실행(RCE)으로 이어질 위험이 있음.
- **설명**: Apache ActiveMQ 5.14.0 이전 버전의 Fileserver 웹 애플리케이션에서 발생하는 취약점(CVE-2016-3088)으로, HTTP PUT 요청을 이용해 파일을 업로드한 후 MOVE 요청을 통해 실행 가능한 경로로 파일을 이동시킬 수 있음.
- **근거**: curl 명령어를 사용하여 특정 경로에 파일 존재 여부를 확인하는 방식으로 재현 가능함. `curl -X 'GET' -d '' -H 'Host: 10.67.182.168:8161' -H 'User-Agent: Mozilla/5.0 (Fedora; Linux i686; rv:124.0) Gecko/20100101 Firefox/124.0' 'http://10.67.182.168:8161/fileserver/37bqUJvN7ZHqMwVRB5KXFZipYuB.txt'`
- **대응**: Apache ActiveMQ를 취약점이 해결된 버전(5.14.0 이상)으로 즉시 업데이트함.
- **조치**: 사용하지 않는 Fileserver 기능을 비활성화하기 위해 `conf/jetty.xml` 설정 파일에서 관련 웹 애플리케이션 설정을 삭제하거나 주석 처리함.
