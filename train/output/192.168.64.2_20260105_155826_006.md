## Inconsistent Redirection
- **End-Point**: `/login`
- **영향**: 리다이렉션 응답 본문에 포함된 HTML 요소가 피싱 공격에 악용되거나 의도치 않은 정보 노출을 유발할 수 있음.
- **설명**: 서버가 3xx 상태 코드를 반환하며 리다이렉션을 수행할 때, 응답 본문에 예상치 못한 HTML 코드(링크 또는 폼 등)가 포함되어 있는 상태임.
- **근거**: `curl "http://192.168.64.2:5000/login" -e "http://192.168.64.2:5000/" -d "username=alice&password=Letm3in_"` 호출 시 3xx 응답과 함께 HTML 본문이 확인됨.
- **대응**: 리다이렉션 응답 시 본문 내용을 완전히 제거하거나, 표준 사양에 따라 사용자에게 노출되지 않는 최소한의 정보만 포함하도록 구성해야 함.
- **조치**: 웹 애플리케이션 프레임워크 또는 서버 설정에서 리다이렉션 시 자동으로 생성되는 HTML 본문 기능을 비활성화하고 Location 헤더만 전송하도록 수정함.
