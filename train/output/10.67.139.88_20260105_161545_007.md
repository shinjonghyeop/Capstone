## Unencrypted Channels
- **End-Point**: `/cgi-bin/test.cgi`, `/text/css` 외 3개
- **영향**: 데이터 전송 시 암호화되지 않은 평문으로 통신하여 중간자 공격(MITM)을 통한 정보 유출 및 데이터 변조 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션 설정이 존재하지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 처리됨.
- **근거**: `curl "http://10.67.139.88/cgi-bin/test.cgi"` 실행 시 HTTP 프로토콜을 통해 평문으로 접근 가능한 것을 확인함.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉션하도록 서버 설정을 변경하고 HSTS(HTTP Strict Transport Security)를 적용해야 함.
- **조치**: 웹 서버(Apache, Nginx 등) 설정에서 80번 포트의 모든 요청을 443번 포트로 리다이렉트(301 Redirect)하도록 구성하고 보안 인증서를 설치함.
