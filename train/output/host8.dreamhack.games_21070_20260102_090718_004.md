## Unencrypted Channels
- **End-Point**: `/`
- **영향**: 네트워크 패킷 감청을 통해 사용자의 로그인 정보, 세션 쿠키 등 민감 데이터가 노출될 수 있으며 중간자 공격(MITM)에 의한 데이터 변조 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션이 설정되어 있지 않아 모든 HTTP 요청이 암호화되지 않은 평문 상태로 처리됨.
- **근거**: `curl "http://host8.dreamhack.games:21070/"` 명령을 실행했을 때 HTTPS로의 리다이렉션 없이 HTTP 통신이 그대로 허용됨을 확인.
- **대응**: 웹 서버 설정을 수정하여 모든 HTTP(80 포트 등) 요청을 HTTPS(443 포트 등)로 강제 리다이렉션해야 함.
- **조치**: SSL/TLS 인증서를 적용하고 서버 설정에 HSTS(HTTP Strict Transport Security) 헤더를 추가하여 브라우저가 항상 안전한 연결을 사용하도록 강제함.
