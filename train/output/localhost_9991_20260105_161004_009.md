## Unencrypted Channels
- **End-Point**: `/www/XSS/XSS_level2.php`, `/www/XSS/XSS_level1.php` 외 3개
- **영향**: 데이터가 평문으로 전송되어 중간자 공격(MITM)에 노출될 수 있으며, 로그인 정보나 세션 쿠키 등 민감한 데이터가 탈취될 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션 설정이 존재하지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 처리됨.
- **근거**: ```curl "http://localhost:9991/www/XSS/XSS_level2.php"``` 명령을 통해 HTTP 프로토콜로 서비스에 접근 가능한 것이 확인됨.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉트하도록 서버 설정을 변경하고 HSTS(HTTP Strict Transport Security)를 적용함.
- **조치**: 웹 서버(Nginx, Apache 등) 설정 파일에서 80번 포트의 모든 요청을 443번 포트로 리다이렉트(301 Redirect)하도록 구성하고 SSL/TLS 인증서를 적용함.
