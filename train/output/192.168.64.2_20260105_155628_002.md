## Fuzzing Parameters - Cross-Site Scripting
- **End-Point**: `http://192.168.64.2:9991/www/FileInclusion/pages/lvl1.php/`, `http://192.168.64.2:9991/www/XSS/XSS_level2.php/` 외 2개
- **영향**: 공격자는 피해자의 브라우저에서 악성 스크립트를 실행하여 세션 쿠키 탈취, 사용자 세션 하이재킹, 웹 페이지 변조 및 피싱 공격을 수행할 수 있습니다.
- **설명**: GET 요청의 매개변수(order_id, cookie, debug 등)를 통해 전달된 입력값이 서버 응답에 적절한 검증 없이 반사되어 발생하는 Reflected XSS 취약점입니다.
- **근거**: 다수의 매개변수에 XSS 페이로드를 삽입하여 전송했을 때 서버 응답에서 해당 스크립트가 실행됨을 확인하였습니다.

- **대응**: 모든 사용자 입력값에 대해 신뢰할 수 없는 데이터로 간주하고 서버 측에서 엄격한 필터링 및 검증을 수행해야 합니다.
- **조치**: 사용자 입력값을 HTML 응답에 포함할 때 반드시 HTML 엔티티 인코딩을 적용하고, 최신 보안 정책인 Content Security Policy(CSP)를 도입하여 악성 스크립트의 실행을 차단해야 합니다.
