- **영향**: 게시글 본문에 삽입된 악성 스크립트가 해당 페이지를 열람하는 모든 사용자의 브라우저에서 실행되어 세션 탈취, 개인정보 유출, 악성 사이트 리다이렉션 등의 피해를 줄 수 있음.
- **설명**: 게시글 작성 시 입력된 콘텐츠가 서버에 저장된 후, 메인 페이지 등에서 출력될 때 적절한 검증이나 필터링 없이 그대로 브라우저에 렌더링되어 발생하는 Stored XSS 취약점임.
