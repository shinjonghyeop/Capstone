## Information Disclosure - Full Path
- **End-Point**: `/memo`
- **영향**: 서버의 내부 디렉토리 구조가 노출되어 공격자가 시스템 구성을 파악하고 추가적인 공격을 계획하는 데 악용될 수 있음.
- **설명**: `/memo` 엔드포인트의 응답에 `/etc/passwd`와 같은 잠재적인 시스템 경로 정보가 포함되어 있어 내부 정보가 유출됨.
- **근거**: `curl "http://127.0.0.1:8000/memo?memo=hello"` 명령 실행 시 응답에서 시스템 경로 노출 확인.
- **대응**: 운영 환경에서 상세 에러 메시지 출력을 제한하고 시스템 내부 경로가 응답에 포함되지 않도록 설정을 강화함.
- **조치**: 사용자에게 표준화된 에러 페이지를 반환하고 응답 데이터에서 절대 경로를 마스킹하거나 제거하도록 소스 코드를 수정함.
