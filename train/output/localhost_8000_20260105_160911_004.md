## Unencrypted Channels
- **End-Point**: `/`, `/vuln` 외 3개
- **영향**: 암호화되지 않은 통신 채널을 통해 데이터가 전송되므로 중간자 공격(MITM)에 의해 민감 정보가 노출되거나 세션이 탈취될 위험이 있음.
- **설명**: 해당 호스트에 HTTPS 리다이렉션이 설정되어 있지 않아 모든 HTTP 요청이 암호화되지 않은 평문(Clear Text) 상태로 처리됨.
- **근거**: `curl "http://localhost:8000/"` 명령을 통해 HTTP 요청이 성공적으로 처리되며 평문 통신이 이루어짐을 확인.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉트하고 브라우저가 보안 연결을 강제하도록 HSTS(HTTP Strict Transport Security) 설정을 적용함.
- **조치**: 웹 서버 설정에서 80번 포트(HTTP)로 들어오는 모든 요청에 대해 443번 포트(HTTPS)로 301 리다이렉트를 수행하도록 구성함.
