## Inconsistent Redirection
- **End-Point**: `/cgi-bin`
- **영향**: 리다이렉션 응답 본문에 포함된 비정상적인 링크나 폼이 피싱 공격에 악용되거나 사용자 기만을 유도할 수 있음.
- **설명**: HTTP 3xx 리다이렉션 응답 시 본문에 예상치 못한 HTML 데이터(링크, 폼 등)가 포함되어 반환되는 취약점임.
- **근거**: `curl "http://10.67.139.88/cgi-bin"` 명령 실행 시 3xx 응답과 함께 의도하지 않은 HTML 본문이 반환됨을 확인.
- **대응**: 리다이렉션 응답 시 본문 내용을 제거하거나 표준을 준수하는 최소한의 정보만 포함하도록 로직을 점검함.
- **조치**: 서버 설정 또는 애플리케이션 코드를 수정하여 3xx 상태 코드 반환 시 HTTP Body를 비우거나 불필요한 HTML 노출을 차단함.
