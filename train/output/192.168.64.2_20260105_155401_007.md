## Unencrypted Channels
- **End-Point**: `/www/SQL/sql1.php`, `/www/FileUpload/fileupload1.php` 외 8개
- **영향**: 모든 HTTP 통신이 평문으로 전송되어 네트워크 스니핑을 통한 데이터 유출 및 중간자 공격(MITM)에 취약함.
- **설명**: 해당 호스트에 HTTPS 리다이렉션 설정이 존재하지 않아 모든 HTTP 요청이 암호화되지 않은 상태로 서비스됨.
- **근거**: `curl "http://192.168.64.2:9991/www/SQL/sql1.php"` 명령을 통해 암호화되지 않은 HTTP 프로토콜로 자원에 접근 가능한 것을 확인.
- **대응**: 모든 HTTP 요청을 HTTPS로 강제 리다이렉션하도록 서버 설정을 변경하고 유효한 SSL/TLS 인증서를 적용함.
- **조치**: 웹 서버(Apache, Nginx 등) 설정에서 80번 포트 요청을 443번 포트로 전달하는 규칙을 구성하고 HSTS(HTTP Strict Transport Security) 헤더를 적용함.
